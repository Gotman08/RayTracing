Description détaillée

Union qui stocke les donnees concretes d'un objet geometrique.

Comme on ne peut pas utiliser le polymorphisme dynamique sur GPU, on utilise une union pour stocker soit une Sphere, soit un Plane dans le meme espace memoire. Un seul des deux membres est valide a la fois, selon le type indique dans le HittableObject parent.
