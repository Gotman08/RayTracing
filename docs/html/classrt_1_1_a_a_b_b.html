Description détaillée

Boite englobante alignee aux axes (Axis-Aligned Bounding Box)

Une AABB est definie par trois intervalles (un par axe x, y, z) qui forment un parallelepipede rectangle aligne avec les axes du repere. Grace a cet alignement, le test d'intersection avec un rayon est tres rapide (methode des slabs). Utilisee dans le BVH pour organiser la scene en hierarchie de volumes.

◆ hit()

__host__ __device__ bool rt::AABB::hit ( const Ray & r,
Interval ray_t ) const
inline

Test d'intersection rayon-boite par la methode des slabs.

Pour chaque axe, on calcule les parametres t d'entree et de sortie du rayon dans la tranche (slab) definie par l'intervalle de cet axe. L'intersection existe si et seulement si les intervalles de t se chevauchent sur les trois axes simultanement. C'est un algorithme tres efficace et adapte au GPU.

Paramètres
rLe rayon a tester
ray_tL'intervalle de validite pour le parametre t du rayon
Renvoie
true si le rayon intersecte la boite dans l'intervalle donne

◆ surface_area()

__host__ __device__ float rt::AABB::surface_area ( ) const
inline

Calcule l'aire de la surface de la boite.

La formule est 2*(dx*dy + dy*dz + dz*dx), c'est-a-dire la somme des aires des 6 faces. Utilisee dans l'heuristique SAH (Surface Area Heuristic) pour optimiser la construction du BVH.

Renvoie
L'aire totale de la surface de la boite
