Description détaillée

Represente un intervalle ferme [min, max] de nombres flottants.

Cette classe offre des operations courantes sur les intervalles : test d'appartenance, clamping, union de deux intervalles, etc. Par defaut, un intervalle est vide (min = +infini, max = -infini).

Restreint une valeur a l'intervalle [min, max].

Si x < min, retourne min. Si x > max, retourne max. Sinon retourne x. Tres utile pour clamper les couleurs avant la conversion en entier 8 bits.
