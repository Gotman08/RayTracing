Description détaillée

Couche de compatibilite pour compiler le code sans le toolkit CUDA.

Ce fichier permet de compiler le meme code source aussi bien avec le compilateur CUDA (nvcc) qu'avec un compilateur C++ standard. Lorsque CUDACC n'est pas defini (compilation sans CUDA), les macros specifiques a CUDA (host, device, global, constant) sont definies comme vides, et une declaration forward de curandState est fournie. Cela permet au code de rester syntaxiquement valide meme sans le runtime CUDA, ce qui est utile pour le rendu CPU et pour les tests unitaires.
