Description détaillée

Structure contenant tous les parametres de configuration du rendu.

Cette structure regroupe les parametres essentiels pour configurer le rendu : la resolution de l'image, le nombre d'echantillons par pixel (pour l'antialiasing), la profondeur maximale de recursion des rayons, et le choix entre un ciel degrade ou une couleur de fond uniforme. Les valeurs par defaut donnent un bon compromis qualite/performance.

Définition à la ligne 26 du fichier render_config.cuh.
