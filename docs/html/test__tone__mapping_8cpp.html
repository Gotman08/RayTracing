Description détaillée

Tests unitaires pour le tone mapping Reinhard et la correction gamma.

Ce fichier verifie le bon fonctionnement de la conversion HDR vers LDR via l'operateur de tone mapping Reinhard, ainsi que la correction gamma appliquee avant l'affichage. On teste les cas limites (noir, blanc), les valeurs elevees, la monotonie, et le clamping des valeurs negatives.
