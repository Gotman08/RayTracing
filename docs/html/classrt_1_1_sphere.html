Description détaillée

Represente une sphere dans la scene 3D.

La sphere est definie par son centre, son rayon et un materiau. Elle herite de Hittable et implemente la methode hit() qui resout l'equation quadratique d'intersection rayon-sphere. La boite englobante est calculee automatiquement dans le constructeur.

__host__ __device__ bool rt::Sphere::hit ( const Ray & r,
Interval ray_t,
HitRecord & rec ) const
inline

Teste l'intersection entre un rayon et cette sphere.

On resout l'equation quadratique issue de la substitution de l'equation parametrique du rayon P(t) = O + t*D dans l'equation de la sphere |P - C|^2 = r^2. On utilise la formule optimisee avec h = dot(D, OC) pour eviter le facteur 2. Si le discriminant est negatif, il n'y a pas d'intersection. Sinon, on prend la racine la plus proche qui se trouve dans l'intervalle valide. Les coordonnees UV sont calculees pour le placage de texture.

Paramètres
rLe rayon a tester.
ray_tIntervalle de validite du parametre t.
recStructure de sortie remplie avec les informations d'intersection.
Renvoie
true si une intersection valide est trouvee, false sinon.
