Description détaillée

Generateur de nombres aleatoires pour le rendu CPU.

Ce fichier fournit la classe CPURandom qui encapsule le generateur mt19937 de la bibliotheque standard, ainsi que des fonctions utilitaires pour generer des vecteurs aleatoires utilises dans le lancer de rayons (directions diffuses, echantillonnage de disque, hemisphere, etc.).
