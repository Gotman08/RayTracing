Description détaillée

Point d'entree principal de la suite de tests unitaires du raytracer.

Ce fichier contient la fonction main() qui orchestre l'execution de toutes les suites de tests : Vec3, Ray, Interval, Sphere, Plane, AABB, Materials, Camera, Tone Mapping et BVH. Il affiche un recapitulatif des resultats a la fin et retourne un code d'erreur si des tests ont echoue.

Documentation des fonctions

◆ main()

int main ( )

Fonction principale : execute toutes les suites de tests et affiche le bilan.

Chaque suite de tests est lancee l'une apres l'autre. A la fin, un recapitulatif indique le nombre de tests passes et echoues. Le programme retourne 0 si tous les tests ont reussi, ou 1 si au moins un test a echoue.

Renvoie
0 si tous les tests passent, 1 sinon.
