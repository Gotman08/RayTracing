Description détaillée

Dispatch de l'interaction rayon-materiau selon le type de materiau.

Ce fichier contient les fonctions de dispatch qui redirigent l'appel de diffusion (scatter) vers la bonne implementation selon le type du materiau (lambertien, metal ou dielectrique). Il existe une version GPU (scatter) et une version CPU (scatter_cpu) qui utilisent des generateurs aleatoires differents.
