Description détaillée

Classe de base pour tous les objets de la scene pouvant etre intersectes par un rayon.

Cette classe stocke le type de l'objet et sa boite englobante (AABB). Les classes derivees (Sphere, Plane) heritent de Hittable et implementent leur propre methode hit() pour calculer l'intersection avec un rayon. Compatible host (CPU) et device (GPU) grace aux qualificateurs CUDA.
