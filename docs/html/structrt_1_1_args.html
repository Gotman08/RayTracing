Description détaillée

Structure regroupant tous les arguments en ligne de commande.

Contient les parametres de rendu (resolution, samples, profondeur), les options de sortie, le choix du mode CPU/GPU, ainsi que les parametres du mode interactif (controles camera, accumulation). Chaque champ possede une valeur par defaut raisonnable.
