Description détaillée

Tests unitaires pour la classe Ray (rayon)

Ce fichier verifie le bon fonctionnement de la classe Ray qui represente un rayon dans l'espace 3D, defini par une origine et une direction. On teste les constructeurs (par defaut, parametre, avec temps), les accesseurs (origin, direction, time) et la methode at(t) qui calcule le point le long du rayon a la distance parametrique t.
