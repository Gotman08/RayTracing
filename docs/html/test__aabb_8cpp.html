Description détaillée

Tests unitaires pour la classe AABB (Axis-Aligned Bounding Box)

Ce fichier teste la boite englobante alignee sur les axes, structure fondamentale pour l'acceleration du raytracing via les BVH (Bounding Volume Hierarchy). On verifie les differents constructeurs (points, intervalles, fusion), les tests d'intersection rayon-boite, le calcul du centroide, de l'aire de surface, de l'axe le plus long et l'acces aux intervalles par axe.
