Description détaillée

Structure d'acceleration par hierarchie de volumes englobants.

Le BVH organise les primitives de la scene dans un arbre binaire de boites englobantes. Cela permet de rejeter rapidement les rayons qui ne touchent pas un groupe de primitives, ce qui accelere enormement le rendu par rapport a un test d'intersection avec chaque primitive. La structure est stockee sous forme de tableau plat (pas de pointeurs) pour etre compatible avec la memoire GPU.

Traversee iterative du BVH pour trouver l'intersection la plus proche.

Utilise une pile locale (tableau de taille fixe 64) au lieu de la recursion pour etre compatible GPU. Parcourt l'arbre en profondeur et teste l'intersection avec les boites englobantes avant de tester les primitives, ce qui permet d'eliminer rapidement les branches inutiles.

Paramètres
rLe rayon a tester
ray_tIntervalle de distance valide pour l'intersection [tmin, tmax]
recEnregistrement d'intersection rempli si un hit est trouve
Renvoie
true si le rayon intersecte au moins une primitive, false sinon
