Description détaillée

Noeud de l'arbre BVH (Bounding Volume Hierarchy)

Chaque noeud contient une boite englobante (AABB) qui enveloppe tous ses enfants. Un noeud est soit interne (avec deux enfants left/right), soit une feuille (avec un indice vers une primitive).
