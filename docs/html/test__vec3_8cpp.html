Description détaillée

Tests unitaires pour la classe Vec3 (vecteur 3D)

Ce fichier contient l'ensemble des tests unitaires permettant de valider le bon fonctionnement de la classe Vec3. On y verifie les constructeurs, les operateurs arithmetiques (+, -, *, /), les operateurs composes (+=, *=, /=), ainsi que les fonctions mathematiques essentielles au raytracing : produit scalaire (dot), produit vectoriel (cross), reflexion, refraction, normalisation et detection de vecteurs quasi-nuls. Les alias Point3 et Color sont egalement testes.
