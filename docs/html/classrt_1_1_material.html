Description détaillée

Represente un materiau generique attache a un objet de la scene.

Cette classe stocke toutes les proprietes physiques d'un materiau : le type (lambertien, metal ou dielectrique), la couleur (albedo), le facteur de flou pour les metaux (fuzz) et l'indice de refraction pour les dielectriques (ior). Elle est utilisable a la fois sur le CPU (host) et le GPU (device) grace aux qualificateurs CUDA.

Retourne la lumiere emise par le materiau.

Par defaut, un materiau n'emet pas de lumiere (retourne noir). Cette methode peut etre surchargee pour creer des materiaux emissifs (sources de lumiere dans la scene).
