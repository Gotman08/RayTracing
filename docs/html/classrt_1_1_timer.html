Description détaillée

Chronometre CPU pour mesurer le temps d'execution des differentes etapes.

Cette classe permet de chronometrer plusieurs etapes successives et de les enregistrer avec un nom. A la fin, on peut afficher un rapport complet avec la duree de chaque etape et son pourcentage du temps total. Elle utilise std::chrono::high_resolution_clock pour une precision maximale.
