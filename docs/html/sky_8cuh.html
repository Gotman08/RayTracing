Description détaillée

Modele de ciel avec gradient et soleil optionnel.

Ce fichier definit la classe Sky pour simuler un ciel avec un degrade entre l'horizon et le zenith, ainsi qu'un disque solaire optionnel. Deux fonctions libres sont aussi fournies : sky_gradient() pour un ciel bleu classique et sky_black() pour un fond noir.
