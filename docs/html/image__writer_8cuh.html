Description détaillée

Sauvegarde du frame buffer en fichier image.

Ce fichier fournit une fonction utilitaire pour exporter le resultat du rendu (frame buffer) vers differents formats d'image : PNG, JPG, BMP, TGA (via stb_image_write) ou PPM (ecriture manuelle). La conversion des couleurs flottantes vers des octets est parallelisee avec OpenMP pour de meilleures performances.
