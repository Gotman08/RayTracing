Description détaillée

Tests unitaires pour l'intersection rayon-plan.

Ce fichier teste la methode hit() de la classe Plane qui calcule l'intersection entre un rayon et un plan infini. On verifie les cas classiques : intersection perpendiculaire, oblique, rayon parallele au plan, rayon pointant dans la direction opposee, rayon venant de derriere, contraintes d'intervalle de t. On teste aussi les normales (face avant/arriere, unitaire) et des configurations geometriques variees (plan vertical, plan diagonal, plan decale).
