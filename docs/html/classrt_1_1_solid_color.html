Description détaillée

Texture de couleur unie.

Cette classe represente une texture qui retourne toujours la meme couleur (l'albedo). C'est utile pour les materiaux de couleur uniforme comme un mur peint ou une balle de couleur. L'albedo represente la fraction de lumiere reflechie par la surface.
