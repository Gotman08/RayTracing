Description détaillée

Vecteur 3D a composantes flottantes.

Classe de base du raytracer servant a representer des positions (Point3), des directions ou des couleurs (Color) dans l'espace 3D. Tous les operateurs arithmetiques courants sont surcharges pour faciliter les calculs vectoriels. Compatible host/device CUDA.
